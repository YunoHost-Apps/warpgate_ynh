- Le nom d'utilisateur par défaut est `admin` et le mot de passe est celui que vous avez défini lors de l'installation.

- Si votre serveur ynh est protégé par un pare-feu, vous devez ouvrir les ports suivants pour tous les services que vous souhaitez utiliser : `SSH: __PORT_SSH__`, `MySQL: __PORT_MYSQL__`, `Postgres: __PORT_POSTGRES__`, `Kubernetes: __PORT_KUBERNETES__`
