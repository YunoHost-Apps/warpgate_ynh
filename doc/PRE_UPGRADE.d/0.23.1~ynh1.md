Two significant changes to this package as of version 0.23.1:
- the config file (`__INSTALL_DIR__/warpgate.yaml`) will now be updated regularly to the latest version so as to include new variables. If you made custom changes to yours, please review the updated file (`https://github.com/YunoHost-Apps/warpgate_ynh/blob/main/conf/nginx.conf) prior to upgrading.
- Kubernetes support has been added by upstream. It is labeled *experimental*.

If you experience problems with either, please file a ticket here: https://github.com/YunoHost-Apps/warpgate_ynh/issues
