Deux changements importants ont été apportés à ce paquet à partir de la version 0.23.1 :

- Le fichier de configuration (`__INSTALL_DIR__/warpgate.yaml`) sera désormais mis à jour régulièrement avec la dernière version afin d'y inclure les nouvelles variables. Si vous avez apporté des modifications personnalisées à votre fichier, veuillez consulter la version mise à jour (`https://github.com/YunoHost-Apps/warpgate_ynh/blob/main/conf/nginx.conf`) avant de procéder à la mise à niveau.

- La prise en charge de Kubernetes a été ajoutée par l'équipe en amont. Elle est actuellement expérimentale.

Si vous rencontrez des problèmes avec l'un ou l'autre de ces éléments, veuillez ouvrir un ticket ici : https://github.com/YunoHost-Apps/warpgate_ynh/issues
